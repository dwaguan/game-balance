# v1.0 — initial release

Core 2D physics bar-balancing game.
- Rigid bar pivoting about its center, decaying mass over time (difficulty ramp)
- Two balls, 2-player local (P1: A/D/W, P2: ←/→/↑) or vs a heuristic AI
- Invisibility active skill with cooldown
- Win when opponent falls off the bar

Open `index.html` to play this version.
