# v1.4 — skill-box mode

Optional power-up mode.
- Toggle "Skill boxes" in the menu for a ~1.5× longer bar with random pickups
- 4 timed buffs: M heavier mass, A more accel, C shorter invis CD, I inelastic collisions
- Touch-to-absorb boxes; spawn/collect/respawn lifecycle
- Buff halo + timer ring on buffed balls

This is the latest version (matches the repo root).
Open `index.html` to play this version.
