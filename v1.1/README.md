# v1.1 — phone support

Play on touch devices.
- On-screen hold buttons (◀ / INVIS / ▶)
- Optimized phone layout: fills viewport, no scroll, thumb-reachable controls
- Tilt (gravity sensor) steering with calibration + live indicator
- Touch supports vs-AI only (2-player needs a keyboard)

Open `index.html` to play this version.
